# Computer Graphics Project — 3D Soccer Field in Godot

This repository contains a project developed for the second assessment of the Computer Graphics course at the Federal University of Alagoas (UFAL), taught by **Prof. Dr. Marcelo Costa Oliveira**.

## Project Overview

The goal of this project is to extend the [soccer field project](https://github.com/lawtherea/soccer-cg-ab1) developed during the first assessment by improving its visual quality through 3D modeling, lighting, textures, and scene refinement.

For this second assessment, the project was rebuilt and refined using **Godot**, focusing on a more realistic three-dimensional soccer environment. The scene includes a modeled soccer field, stadium elements, textured objects, lighting setup, and visual improvements intended to increase realism and immersion.

## Assignment Objective

The main objective of the second assessment is to extend the AB1 project using lighting and texture techniques, applying computer graphics concepts in a 3D environment.

## Assignment Requirements

The project was developed based on the following requirements:

* Use OpenGL or a Game Engine, such as Unity, Unreal Engine, Godot, or similar;
* Create the lighting of the modeled scene, exploring different light sources, geometric aspects, and lighting models;
* Apply texture mapping to the objects in the scene;
* Refine the modeling created during the first assessment;
* Improve the realism and visual quality of the environment, since realism represents an important part of the final grade.

## Technologies Used

This project was built with:

* **Godot**
